# 🎵 MusicFM

> 君だけのプレイリストを友達と共有しよう

**MusicFM** は、YouTubeの動画URLを改行で入力するだけで、自分だけのプレイリストを再生・共有できるWebアプリです。

## 🔧 機能

- YouTube/ニコニコ動画/SoundCloudの連続再生
- URLリストの読み込み・全消去
- プレイリストの共有リンク生成
- 共有履歴の保存・削除

## 🚀 アクセス

[https://musicfm.pages.dev](https://musicfm.pages.dev)

## 🛠 技術

- SvelteKit
- Tailwind CSS
- YouTube IFrame Player API
- LocalStorage
- Imgur API
